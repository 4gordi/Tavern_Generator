# Tavern_Generator

Рабочая версия: http://tentaculus.ru/tavern/

Генератор создает: 
  * Название таверн
  * Расу/Имя/Привычки и прочие особенности тавернщика
  * Блюда и расценки
  * Атмосферу и случайные события
  
  и другое...
